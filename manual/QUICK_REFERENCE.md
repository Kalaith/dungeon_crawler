# Quick Reference: Merged RoA + D&D 5e System

## Attribute Modifiers
```
Modifier = (Attribute - 10) / 2 (rounded down)

Attribute | Modifier
----------|----------
8-9       | +0
10-11     | +0
12-13     | +1
14-15     | +2
16-17     | +3
18-19     | +4
```

## Skill Checks
```
Skill Check = d20 + Attribute Modifier + Proficiency (if trained)

DC Table:
Very Easy: 5
Easy: 10
Medium: 15
Hard: 20
Very Hard: 25
Nearly Impossible: 30
```

**Advantage:** If secondary attribute modifier is +2 or higher, roll 2d20 and take higher.

## Combat

### Attack Rolls
```
Attack Roll = d20 + Attribute Modifier + Proficiency + Weapon Skill

Melee Weapons: Use ST modifier
Finesse Weapons: Use ST or DX modifier (your choice)
Bows: Use AG modifier
Thrown/Crossbows: Use DX modifier
```

### Damage
```
Damage = Weapon Dice + Attribute Modifier

Examples:
Dagger: 1d4 + modifier
Shortsword: 1d6 + modifier
Longsword: 1d8 + modifier
Greatsword: 2d6 + modifier
Longbow: 1d8 + modifier
```

**Critical Hit:** Natural 20 = Roll weapon dice twice + add modifier once

**Two-Handed Weapons:** Add +2 damage

### Initiative
```
Initiative = d20 + AG Modifier + IN Modifier (if IN ≥ 12)
```

### Armor Class (AC)
```
AC = 10 + Armor Bonus + AG Modifier (if wearing light/medium armor)

Light Armor: AC 11-12 + full AG modifier
Medium Armor: AC 13-15 + AG modifier (max +2)
Heavy Armor: AC 16-18 (no AG modifier)
Shield: +2 AC
```

## Magic System

### AP Pool
```
Base AP = 10 + (Intelligence Modifier × 2) + Class Bonus

Class AP Bonuses:
Wizard/Magician: +20 AP
Cleric/Druid: +15 AP
Warlock/Sorcerer: +12 AP
Bard: +10 AP
Paladin: +8 AP
Ranger: +5 AP
```

### Spell Costs
```
Cantrip: 0 AP
Level 1: 5 AP
Level 2: 8 AP
Level 3: 12 AP
Level 4: 18 AP
Level 5: 25 AP
Level 6: 35 AP
Level 7: 45 AP
Level 8: 60 AP
Level 9: 80 AP
```

### Spell Save DC & Attack
```
Spell Save DC = 8 + Proficiency + Spellcasting Modifier
Spell Attack Bonus = Proficiency + Spellcasting Modifier

Spellcasting Attributes:
Wizard/Magician: Intelligence (IT)
Cleric/Druid/Ranger: Wisdom (WD)
Warlock: Intuition (IN)
Sorcerer/Bard/Paladin: Charisma (CH)
```

### AP Recovery
- **Long Rest (8 hours):** Restore all AP
- **Short Rest (1 hour):** Restore 25% of max AP (once per day)
- **Meditation (Magicians only):** Convert 5 HP to 5 AP

## Proficiency Bonus by Level
```
Level 1-4:   +2
Level 5-8:   +3
Level 9-12:  +4
Level 13-16: +5
Level 17-20: +6
```

## Saving Throws
```
Strength Saves: Use ST modifier
Dexterity Saves: Use AG modifier
Constitution Saves: Use CO modifier
Intelligence Saves: Use IT modifier
Wisdom Saves: Use WD modifier
Charisma Saves: Use CH modifier
```

## Negative Attributes (Trigger at 6+)
```
Superstition (SN): Disadvantage on saves vs. illusion/enchantment
Acrophobia (AC): WD save DC 15 at heights or become Frightened
Claustrophobia (CL): WD save DC 15 in tight spaces or become Frightened
Avarice (AV): WD save DC 15 when seeing treasure or must take it
Necrophobia (NE): CR save DC 15 vs. undead or become Frightened
Curiosity (CU): WD save DC 15 vs. mysteries or must investigate
Violent Temper (VT): WD save DC 15 when insulted or must attack
```

**Frightened Condition:**
- Disadvantage on ability checks and attack rolls
- Cannot move closer to source of fear
- Lasts until you leave situation or succeed on save

## Combat Actions (Your Turn)
- **Attack:** Make weapon attack
- **Skill:** Use class ability
- **Spell:** Cast spell (costs AP)
- **Item:** Use potion/consumable
- **Switch Row:** Move Front ↔ Back (free action)
- **Defend:** Gain Advantage on saves, enemies have Disadvantage

## Row Positioning
```
Front Row:
- Take normal damage
- Deal normal melee damage
- Best for: Warriors, Paladins, melee fighters

Back Row:
- Melee attacks vs you have Disadvantage
- Your melee attacks have Disadvantage
- Ranged/Magic: Normal
- Best for: Mages, Clerics, Rangers, Archers
```

## Spell Learning

### Prepared Casters (Cleric, Druid, Paladin)
- Know all spells on class list
- Prepare: Spellcasting Modifier + Level spells per day
- Change prepared spells after long rest

### Known Casters (Wizard, Sorcerer, Warlock, Bard, Ranger)
```
Class          | Spells at Level 1 | Per Level
---------------|-------------------|----------
Wizard/Magician| 6 spells          | +2 spells
Sorcerer       | 4 spells          | +1 spell
Warlock        | 2 spells          | +1 spell
Bard           | 4 spells          | +1 spell
Ranger         | 2 spells (Lv 2+)  | +1 spell
```

## Common Formulas Summary
```
Attribute Modifier = (Attribute - 10) / 2
Skill Check = d20 + Attribute Mod + Proficiency
Attack Roll = d20 + Attribute Mod + Proficiency + Weapon Skill
Damage = Weapon Dice + Attribute Mod
Initiative = d20 + AG Mod + IN Mod (if IN ≥ 12)
AC = 10 + Armor + AG Mod (if applicable)
Spell Save DC = 8 + Proficiency + Spellcasting Mod
Spell Attack = Proficiency + Spellcasting Mod
AP Pool = 10 + (IT Mod × 2) + Class Bonus
```

---

**Remember:** 
- Natural 20 on attack = Critical Hit (double weapon dice)
- Natural 1 on attack = Critical Miss (automatic failure)
- Advantage = Roll 2d20, take higher
- Disadvantage = Roll 2d20, take lower
- Concentration = Only one concentration spell at a time
